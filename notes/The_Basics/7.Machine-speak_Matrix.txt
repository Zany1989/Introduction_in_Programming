7. Машинный язык и Матрица.

Итак, помните фильм «Матрица» с Киану Ривзом, который только что снялся в «Невероятных приключениях Билла и Теда»?
В фильме «Матрица» люди понимают, что на самом деле живут в симуляции вселенной, симуляции, которая, по сути,
накладывается им на глаза, чтобы отвлечь от того факта, что на самом деле они живут в очень грязной ретро-панк вселенной,
где роботы контролируют мир и используют людей в качестве батареек. Многие вещи, показанные в фильме, работают аналогично
и в реальности.

У компьютеров есть свой собственный язык, и этот язык называется машинным кодом. И, как и в фильме «Матрица», машинный
код невероятно сложен для нас, людей, не только для написания, но и для чтения и даже понимания. Только самые настоящие
зануды, настоящие гики, способны читать и писать подобные вещи, и это невероятно впечатляющее достижение, заслуживающее
восхищения. Когда компьютер хочет что-то сделать сам себе, или приложение, или периферийное устройство, например мышь,
хочет, чтобы ему что-то сказали, оно делает это с помощью различных вариантов машинного кода.

Материнская плата компьютера содержит процессор, оперативную память, постоянную память и жесткий диск. Центральный
процессор компьютера - CPU (ЦП) состоит из множества металлических кремниевых квадратов, транзисторов. Они работают
следующим образом: когда через транзистор проходит электрический ток, он равен 1, когда электрический ток через транзистор
не проходит, он равен 0. Необходимо помнить, основной принцип работы транзистора таков: включено/выключено, on/off, 1/0.

Машинный код сложен для понимания человеком, но этот неэффективный, многословный код на самом деле обладает преимуществом
бесконечной настраиваемости. Что означает бесконечная настраиваемость? Это означает, что, поскольку вам приходится тратить
много времени на детальную проработку каждой части отправляемого сообщения, это дает вам больше контроля над тем, что вы
можете сказать. Почему машинный код так сложно понять человеку? Потому что последовательности чисел 1 и 0 могут быть очень
длинными. Чем сложнее команда, тем длиннее машинный код. Чем больше файл по объему памяти, тем длиннее его двоичный код.

 - Скорость процессора (GHz).

Скорость компьютерного процессора измеряется в герцах (Гц) и обозначает количество циклов в секунду, которые может
выполнить процессор. Современные процессоры обычно работают на частотах от 1 ГГц (гигагерц) до 3,8 ГГц, а более мощные
модели способны достигать ещё более высоких скоростей.

Один такой цикл — это включение или выключение одного транзистора, 1 или 0. Один транзистор включается на миллионную долю
секунды.

ГГц, сокращение от гигагерц, — это единица частоты, равная одному миллиарду герц. Она широко используется для измерения
скорости обработки данных компьютером, переменного тока и электромагнитных (ЭМ) частот.

 - Языки программирования.

Языки программирования делятся на три группы: низкого уровня, среднего уровня высокого уровня. Машинный код — это
низкоуровневый код языка программирования. Каждый язык программирования имеет определенный синтаксис — условия, которые
необходимо соблюдать при написании кода.

Существует множество языков программирования. Каждый из них предназначен для разработки различных задач. Где используются
языки программирования? В веб-разработке используется язык программирования JavaScript, реже Python. Язык Python обычно
используется для работы с искусственным интеллектом. Язык программирования C++ используется в программировании для
Windows, а также в разработке драйверов. Язык программирования Swift используется в разработке программ для MacBook. Язык
программирования Java используется для разработки различных финансовых решений, для работы с данными и деньгами.

Были изобретены низкоуровневые языки программирования (ассемблер, C, C# и C++), которые близки к исходному машинному коду.
Они быстрые, предназначены для аппаратного обеспечения, драйверов, программ, где от них требуется очень быстрая работа.
Низкоуровневые языки программирования предназначены для управления компьютерной периферией (устройствами, используемыми с
компьютером: клавиатурой, монитором, мышью, принтером). Они взаимодействуют с компьютером, используя языки
программирования более высокого уровня, чем машинный код. Для их корректной работы требуются различные драйверы,
обеспечивающие совместимость с компьютером, интерфейс на разных языках и выполнение функций.

Для облегчения понимания, чтения и написания программного кода появились высокоуровневые языки программирования, которые
легко читать, понимать и писать. К ним относятся: JavaScript, Python, PHP, Java, Go, Flutter и другие языки
программирования. Высокоуровневые языки программирования используются для разработки веб-страниц, сервисов и продуктов.
