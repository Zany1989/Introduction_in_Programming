3. Печать.

Объяснение концепции печати.

Печать — это команда, которую можно добавить в код, чтобы указать ему вывести результат выполняемого кода на экран.

Если вы хотите, чтобы веб-сайт выполнил вычисление, недостаточно просто написать для этого код; необходимо «вывести»
результат на экран, чтобы пользователь увидел фактический результат.

В каждом языке программирования существует свой специфический синтаксис для печати. Например, в Python используется
функция print(), в JavaScript — console.log(), а в Java — System.out.println().

Пример в Python:
```python
x = 5
y = 10
result = x + y
print(result)  # Это выведет 15 на экран
```

Пример в JavaScript:
```javascript
let x = 5;
let y = 10;
let result = x + y;
console.log(result); // Это выведет 15 в консоль
```

Печать является важной частью программирования, так как она позволяет разработчикам видеть результаты своих вычислений и
отлаживать код. Без возможности печати было бы сложно понять, что происходит внутри программы и какие данные
обрабатываются. Печать также используется для отображения информации пользователю, такой как сообщения об ошибках,
результаты вычислений или просто для информирования о том, что программа работает.
